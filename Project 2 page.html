Inventory Management Robot Project Overview



Introduction of the Project

The use of robots in warehouses is part of a larger trend towards automation in the logistics and supply chain industries, helping to make these operations more efficient and effective.

This project focuses to design and build robots that can efficiently navigate through a warehouse environment, performing tasks such as inventory management, order fulfillment, and object transportation, with the ultimate goal of creating a robot that can streamline operations and improve productivity in a real-world warehouse setting.

Project Objective

  1. This project aims to design and build a inventory management robot.

  2. The project aims to create an efficient and accurate inventory management system that reduces human error, minimizes inventory discrepancies, and streamlines the inventory restocking process.
  3. The project aims to identify the cargo that needs to be delivered in the form of barcodes.

Key Features

  • Autonomous navigation of robot to place a cargo in a specific predetermined location via preprogrammed instruction.

  • Manual control of robot to load and unload cargo.

  • Incorporation of sensors to detect and avoid obstacles, ensuring safety of the robot.

  • Utilization of barcode scanner to identify and differentiate the colors of cargo, facilitating proper sorting and placement according to specified requirements.

  • Implement of a gripper mechanism allows the robot to grasp and handle cargo efficiently during the loading process.

  • Integration of a storage system equipped with a conveyor belt and vertical stacking capability provides the robot with the means to manage and organize cargo efficiently.


Advantages of Inventory Management Robot

  • Improve efficiency. The robot combines autonomous navigation with manual control, streamlining cargo handling processes and optimizing efficiency in inventory management tasks.

  • Optimized storage. The implemented storage system, including a conveyor belt and vertical stacking capability, allows for efficient organization and stacking of cargo, optimizing space utilization.

  • Consistency. The robot performs tasks consistently according to pre-programmed instructions or sensor-based data, eliminating variability in actions that could occur due to human factors like fatigue or distraction.
